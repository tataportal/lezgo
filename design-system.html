LEZGO Design System

Variables, colores, tipografia, espaciado y componentes. Todo sharp, solo botones redondeados.

Reglas de Diseno

Border-radius: Solo los botones tienen esquinas redondeadas (8px). Cards, containers, inputs, imagenes y todo lo demas es sharp (0px). Excepciones: avatares (50%), badges/pills (100px), progress bars (4px).
Font-sizes: 8 tokens con saltos de ~30%. Cada tamano es visualmente distinguible del anterior. No hay tamanos intermedios.
Colores: Nunca hardcodear hex. Siempre usar var(--token). Para blanco usar var(--text), para negro usar var(--gray-950).
